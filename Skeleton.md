## 스켈레톤(Skeletion) 이란?
웹 페이지에서 로드 시간이 짧은 것처럼 보이게 하는 몇 가지 방법 중 하나!

데이터가 로드되기 전에 콘텐츠의 자리비움(placeholder)를 표시해서 사용자가 기다리는 시간을 좀 덜 지루하게 느끼게끔 하는 UI

## 스켈레톤 컴포넌트의 장단점
- 장점 : 유저들이 로딩 시간을 조금 덜 지루하게 느낌
- 단점 : 스켈레톤을 화면마다 표시해야 되기 때문에 상대적으로 시간이나 비용이 많이 듦
